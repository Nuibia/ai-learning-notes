# 跟着 AI 学习 AI

这是我的个人 AI 学习与公开输出仓库。

我会把每天真正理解的概念、项目实践后的复盘，以及准备发布到掘金等平台的文章同步到这里。这里保存的是适合公开阅读的学习成果，不保存本机配置、项目私有信息和内部协作资产。

## 学习方法

```text
提出问题
→ 主动回忆
→ AI 讲解与纠偏
→ 项目或案例验证
→ 写学习卡片
→ 整理为公开文章
```
